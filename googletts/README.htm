============================================= Google TTS script for Asterisk ============================================= O que é: Este script permite você integrar o google translate com sua URA. Ele permite ha você, digitar o texto, e em seguida o mesmo é falado pode uma voz clara e audível. ------------ Requisitos ------------ Perl perl-libwww sox mpg123 Acesso a internet. ------------ Instalação ------------ Para instalar, copie o googletts.agi para o diretórioo agi-bin. geralmente fica em: /var/lib/asterisk/agi-bin/ ------------ CentOS 6.3 ------------ yum install perl-libwww-perl perl-IO-Socket-SSL perl perl-libwww mpg123 sox -y ------------------------ Ubuntu 12.04 LTS Server ------------------------ Em breve! ----- Uso ----- O Uso é bem simples: agi(googlcat etts.agi,"text",[language],[intkey],[speed]): This will invoke the Google TTS TEXTO: Texto a ser falado IDIOMA: É o idioma que será usado para se pronunciar o texto. Em Idiomas suportados você verá a lista de códigos suportados. INTKEY: Se for definido, o script irá aguardar pela entrada do usuário. SPEED: É a velocidade com que o texto é falado. O script do Google TTS, armazena em um cache local (por padrão / tmp /) para uso futuro. Parâmetros como idioma padrão, habilitar ou desabilitar o cache e cache diretório podem ser configurados editando o script. -------- Exemplos -------- Exemplo de configuração para extensions.conf exten => 1234,1,Answer() exten => 1234,n,agi(googletts.agi,"se a eleição para prefeito fosse hoje, em quem você votaria?") exten => 1234,n,agi(googletts.agi,"Digite 1 para José Antônio. Digite 2 para Tião do açougue. Ou digite 3 para anular seu voto.") exten => 1234,n,Hangup() Uma simples ura de exemplo [URA] exten => s,1,Answer() exten => s,n,Set(TIMEOUT(digit)=5) exten => s,n,agi(googletts.agi,"Bemvindo a ura de demonstração.") ;;Aguarda pelo digito exten => s,n(start),agi(googletts.agi,"Por favor digite um número.",pt-BR,any) exten => s,n,WaitExten() ;; Fala o número digitado exten => _X,1,agi(googletts.agi,"Você digitou ${EXTEN}. Tente digitar outro porfavor.",pt-BR,any) exten => _X,n,WaitExten() exten => i,1,agi(googletts.agi,"Extensão inválida.") exten => i,n,goto(s,start) exten => t,1,agi(googletts.agi,"Tempo limite esgotado.") exten => t,n,goto(s,start) exten => h,1,Hangup() ------------------- Idiomas suportados ------------------- Afrikaans: af Albanian: sq Amharic: am Arabic: ar Armenian: hy Azerbaijani: az
